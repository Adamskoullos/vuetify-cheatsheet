# Displays & Flex

**Breakpoints**:

- `xs`
- `sm`
- `md`
- `lg`
- `xl`

**Display**: >>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

- `d-{value}`
- `d-{breakpoint}-{value}`

> Value

- `none`
- `inline`
- `inline-block`
- `block`
- `table`
- `table-cell`
- `table-row`
- `flex`
- `inline-flex`

- `flex-{value}`
- `flex-{breakpoint}-{value}`

> Value

- `row-reverse`
- `column`
- `column-reverse`
